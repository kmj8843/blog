# 짧은 팁 모음

> 길게 설명할 필요는 없지만, **나중에 다시 찾기 쉬워야 하는 것들**이 있어요.

에러 메시지 하나, 자주 쓰는 명령어 하나, 헷갈리는 용어 하나처럼요.
이곳은 그런 짧은 조각들을 모아두는 공간이에요.

처음부터 끝까지 읽기보다는 **필요할 때 검색해서 바로 확인**하는 쪽에 더 가까워요.
